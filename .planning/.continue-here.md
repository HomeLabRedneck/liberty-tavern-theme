# BLOCKING CONSTRAINTS — Read Before Anything Else

> Discovered through failure. Do not skip.

- [ ] CONSTRAINT: Wrong layout layer — CSS rules on `.container` / `.discovery-list-container` do NOT fix the banner left gap. Gap lives at `#main-outlet-wrapper` flex gap level.
- [ ] CONSTRAINT: CSS variable resolution — `var(--tavern-brass)` and other `var(--tavern-*)` may not resolve in Discourse theme preview. Use hex literals for critical colors.
- [ ] CONSTRAINT: Double padding-top — Do NOT add `padding-top` to `#main-outlet-wrapper`. Discourse auto-sets `body { padding-top: var(--header-offset) }` via JS.
- [ ] CONSTRAINT: `display: contents` on `.category-box-inner` failed — children didn't respond to grid-column rules (DOM nesting deeper than expected). Use `display: grid` directly on `.category-box-inner` instead.
- [ ] CONSTRAINT: `::before { display: none !important }` on badge — suppresses redundant 16px positioned circle from global rule only. The wax-seal fill comes from `background: var(--category-badge-color) !important` on the element itself, not `::before`. No regression.
